## 개요
오늘날 많은 사람들이 SNS를 통해 정보를 얻고 있습니다. 그중 하나인 블로그를 통해서도 리뷰 관련 글을 많이 찾아볼 수 있는데요.
언제부턴가 대가성 리뷰가 많아지면서 비대가성 리뷰를 찾기가 어려워졌습니다.

이에 **광고가 아닌 실제 리뷰만을 보고 싶은 취지**에서 프로젝트를 진행하게 되었습니다.
